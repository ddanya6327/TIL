# Docker

컨테이너 기반의 오픈소스 가상화 플랫폼

## 컨테이너

코드와 모든 종속성을 패키지화하여 응용 프로그램이 한 컴퓨팅 환경에서 다른 컴퓨팅 환경으로 빠르고 안정적으로 실행되도록 하는 소프트웨어의 표준 단위.

- 간단하고 편리하게 프로그램을 실행 시켜 주는 것.

## 도커 이미지

프로그램을 실행하는데 필요한 설정이나 종속성을 가지고 있으며 도커 이미지를 이용하여 컨테이너를 생성한다.

### 이미지의 구성 요소

- 시작시 실행 될 명령어
  - ex) run hello-world
- 파일 스냅샷
  - 실행 할 파일의 스냅샷
